¡Bienvenido! Esta es una aplicación productiva que integra cuatro módulos funcionales. A continuación, se detalla la tecnología utilizada en cada componente para su evaluación.

1. Gestor de Tareas

Aplicación *To-Do List* que permite añadir, completar y eliminar tareas, incluyendo un sistema de filtrado.

  • Persistencia de Datos: Manejo de la clave `STORAGE_KEY` y `localStorage` para guardar y cargar el estado.
  • Estructura de Datos: Uso de `Array` de objetos para el almacenamiento.
  • Filtrado: Funciones para mostrar tareas Pendientes, Completadas o Todas.
  • Manipulación de DOM: `renderizarTareas()` y funciones auxiliares para actualización.

2. Citas Inspiradoras

Un generador que proporciona frases motivacionales y citas famosas en inglés, traducidas por el usuario.

  • API Externa: Consumo de la API de ZenQuotes (`zenquotes.io`).
  • Proxy CORS: Uso del servicio `api.allorigins.win/raw` para evitar problemas de política de origen cruzado (CORS).
  • Asincronía: Manejo de solicitudes con `fetch`, `async/await` y Promesas.
  • Manejo de Errores: Implementación de `try...catch` para gestionar fallos de red o de API.
Ver Cita

3. Conversor de Divisas

Herramienta para obtener tasas de cambio actualizadas y realizar conversiones de moneda.

  • API Externa: Consumo de una API Pública de Tasas de Cambio (Se asume un servicio similar a ZenQuotes).
  • Asincronía: Uso de Promesas y `fetch` para la gestión de solicitudes.
  • Proxy: Petición gestionada a través de un *proxy* para evitar errores de CORS y proteger claves.
  • Lógica de Negocio: Cálculos de conversión y formateo de monedas.
Probar Conversión

4. Cuestionario Interactivo

Reto rápido de conocimiento con preguntas enfocadas, ofreciendo *feedback* inmediato y puntuación final.

  • Fuente de Datos: Carga de preguntas desde un archivo JSON local (`preguntasTrivia.json`).
  • Asincronía: Uso de `fetch` para cargar el archivo JSON.
  • Lógica de Juego: Implementación de contadores, validación de respuestas y sistema de puntuación.
  • Aleatoriedad: Funcionalidad para seleccionar y mezclar preguntas al azar.
